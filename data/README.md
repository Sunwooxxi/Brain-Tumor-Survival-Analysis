# Data

This folder should contain `clinical.tsv` downloaded from TCGA.

## Download Instructions

1. Go to [GDC Portal](https://portal.gdc.cancer.gov)
2. Search for TCGA brain tumor clinical data
3. Download `clinical.tsv` and place it in this folder

## Note

Raw data is not included in this repository due to TCGA data use policy.  
Please download directly from the GDC portal before running the notebooks.
